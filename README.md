# Arduino MQTT

Estudo sobre o uso de MQTT com ESP8266 e Arduino.

## Instalação

Problema em ler a porta USB no Ubuntu 22.04. Existe um conflito com o pacote `brltty`, para resolver remova este pacote:

```bash
sudo apt remove brltty
```
